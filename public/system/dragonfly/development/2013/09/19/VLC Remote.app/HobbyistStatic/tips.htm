Start automatically with your favourite remote

You'll need to use a saved remote for this one. Tap on the 'Add computer' and add the remote that you want to use.

Now just select the 'Use as default remote' option.

When you run VLC remote, it will automatically switch to this remote as soon as VLC is detected running on that machine.

Access Fullscreen from the main page

Turning on fullscreen mode when watching movies is really handy. You can do that with the icon in the main remote page. It's the one that looks like four arrows point outwards.

How to set the default directory for browsing

(requires paid version)
This will only work on a saved remote, so make sure you add the computer you want to control as a saved remote (press 'add computer' then follow the prompts).
Open the remote and browse to the directory you want as a default.
Press the 'default' button and your new default is set.

If you exit VLC remote, then launch it again, you'll start browsing in this directory.

How to access external drives

(requires paid version)

Unfortunately, VLC doesn't recognise aliases or shortcuts (it just tries to play them), so these won't help you.

However you can access your external drives.
Windows: Just hit the up button until you get to the top of the browsing tree. You should see your drives listed there. If not, then you'll need to make sure that your external drive is mapped to a drive letter.
Mac: Navigate to the top directory (just hit the up button until you get there) then look in 'Volumes'. You should see external drives there.

tip: you can set your external drive as your default directory to make browsing faster in future.

Add a whole folder to your playlist

(requires paid version)

just tap on the 'play icon' next to the folder and all its contents will be added to the playlist.
